# BombParty Bot

A little program that plays the Bomb Party web game from [https://jklm.fun](https://jklm.fun/) using Selenium

# What is Bomb Party

Bomb Party is a little online word game hosted on [jklm.fun](https://jklm.fun) (no registration needed). You can easily play with friends or with random people.  

<p align="center">
   <img src="https://user-images.githubusercontent.com/52866589/184105620-9b24e3fa-3d23-4783-b6e1-9769e7daae63.png" width=70% height=70%>
</p>

As you can see on the screen there is a bomb in the middle with a syllable on it. When the bomb is pointing at you you have to type a word containing the syllable as fast as possible. At some random point the bomb will explode and if it is pointing at you at that moment you’ll lose one life. 

# Requirements

- [Chrome](https://www.google.com/intl/en_us/chrome/) or [Chromium](https://www.chromium.org/getting-involved/download-chromium/)
- Python ≥ 3.6

# Installation

First install the required packages 

```bash
pip3 install -r requirements.txt 
```

Then you can do 

```bash
python3 main.py
```

# Usage

Go to [jklm.fun](https://jklm.fun), chose a username (right upper corner) and create a room (Private or Public).

After the room is created you can start the bot using the instructions above.  

Once you’ve entered the link of the room you can clic “Start” to start the bot. 

<p align="center">
   <img src="https://user-images.githubusercontent.com/52866589/184105788-c80d657e-4b07-4599-a9d3-c2760f59d42a.png" width=25% height=25%>
</p>

Just wait a few seconds and you should see it in your room. You can then start the game and play as you normally would.
